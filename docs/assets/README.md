## Capture Guidelines

- **Resolution:** 1440×900 for desktop, 375×812 for mobile (iPhone X viewport)
- **Browser:** Chrome or Safari, with no browser UI in the capture
- **Format:** PNG, optimized for web (compress before committing)
- **Naming:** Lowercase, hyphenated filenames as listed above
- **Content:** Use the live site at production URL for authentic captures

## Adding Screenshots

Place files directly in this folder. Reference them from documentation using relative paths:

```markdown
![Homepage hero](docs/assets/homepage-hero.png)
```
