# Changelog

All notable changes to ExpertNetworks.net are documented here.

## March 2026

### Added
- UX/accessibility audit and fixes: ARIA states, touch targets, mobile layout improvements
- News filter state persistence via URL parameters (shareable filtered views)
- 5 new verified 2026 industry signals (GLG ISO 27001, GLG Equality 100, NewtonX partnerships)
- SEO optimizations: schema markup (`@graph` with BreadcrumbList), H1 keyword alignment, canonical URLs, Open Graph types
- Cross-linking improvements between provider profiles, comparison pages, and alternatives pages
- Pricing link added to primary navigation

### Improved
- Compare modal accessibility (aria-hidden toggling)
- Directory filter toggle accessibility (aria-expanded, aria-controls)
- News view toggle accessibility (aria-pressed state)
- Mobile marquee touch targets increased to 44px minimum (WCAG compliance)
- News coverage bar mobile layout (divider handling)
- Internal link count: 8,815 verified references across 58 pages

## February 2026

### Added
- Provider pricing confidence labeling system (verified, inferred, positioned, not disclosed)
- Buyer pathway navigation on homepage and comparison tool
- Industry-specific guides for consulting, private equity, and hedge fund workflows
- Head-to-head comparison pages (6 provider matchups)
- Provider alternatives pages (3 published)

### Improved
- Compare tool preset system expanded to 14 named configurations
- Provider profiles enriched with detailed services, AI platform, and compliance sections
- News feed filtering with category, source type, significance, and time range controls

## January 2026

### Added
- Compare tool v2: dynamic provider selection, URL state sync, diff mode, evidence notes
- Pricing guide with 7 pricing models and provider-level detail
- "What Is an Expert Network?" comprehensive explainer
- Best Expert Networks top-15 ranking page
- Sources & Methodology page with four-tiered evidence framework

## Late 2025

### Added
- Initial site launch with 33 provider profiles
- Networks directory with filtering and search
- Industry intelligence feed
- About, disclaimer, privacy, and verification pages
- JSON-LD structured data across all pages
- Sitemap generation with Astro integration
